Welcome to Blog API!

Endpoints:

  • GET / - Home
  • GET /posts - get all posts
  • GET /posts/<POSTID> - get post by post ID
  • GET /posts//<AUTHORID> - get posts by author ID
  • GET /authors - get all authors
  • POST /posts - create new post and send mail upon creation
  • POST /authors - create new author
  • PUT /posts/<POSTID> - update post by post ID
  • DELETE /posts/<POSTID> - delete post by post ID
  • DELETE /authors/<AUTHORID> - delete author by author ID

Please refer 'requests.rest' file in the repository to make POST and PUT requests
